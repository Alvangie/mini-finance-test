# 💰 MiniFinance

## 📌 Descripción
MiniFinance es un simulador de finanzas personales que permite registrar ingresos y gastos, calcular el saldo disponible y planificar metas de ahorro.

El sistema analiza los hábitos financieros del usuario y muestra resultados de forma visual mediante gráficos y alertas.

## 💡 Idea elegida
Se desarrolló un simulador de presupuesto personal donde el usuario puede:

- Registrar ingresos y gastos
- Clasificarlos por categorías
- Definir metas de ahorro
- Visualizar proyecciones financieras

El objetivo del proyecto es para manejar "finanzas personales".


## 🛠️ Tecnologías utilizadas
- HTML5  
- CSS3  
- JavaScript (Organizado en módulos)  
- Chart.js

### 📊 Gestión de datos
- Registro de ingresos y gastos
- Clasificación por categorías
- Configuración de frecuencia (mensual, semanal, única)

### 🧮 Cálculos
- Conversión automática a valores mensuales
- Cálculo de ingresos, gastos y ahorro mensual  
- Cálculo de tasa de ahorro

### 🎯 Simulación
- Definición de meta de ahorro
- Cálculo del tiempo necesario para alcanzarla
- Generación de datos simulados en modo prueba

### 📈 Visualización
- Tarjetas con resumen financiero dinámico
- Gráfico de evolución del ahorro (Chart.js)
- Actualización del DOM en tiempo real

### ⚠️ Alertas (desafío obligatorio)
- Advertencias visuales si:
  - El ahorro es insuficiente
  - Se supera el gasto
  - No se alcanza la meta
